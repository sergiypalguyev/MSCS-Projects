There are two input fields only: Check amount and Party size.
Check amount only accepts non-negative, numbers, of type double.
Party size only accepts non-negative, numbers, of tiype integeer.

Clicking the CalculateTip button will compute the tip per-person and total amount per-person.
The tip and total fields are non-editable.

Toast:
Toast warnings will appear if the input is invalid or not available. Zero value is also an invalid value.
